# Quick Dental Care - Clinic OS Showcase

A web-based dental clinic management system with real-time chat, message boards, and staff panels.

## Features

- Real-time chatbot for patient communication
- Staff messaging and announcements
- Patient dashboard
- Admin/Staff panels
- Responsive design

## Setup

1. Place files on a web server with PHP support
2. Configure `proxy/config.php` with your database settings
3. Access via `public/index.html`
